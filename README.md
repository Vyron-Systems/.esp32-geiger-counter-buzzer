# esp32-geiger-counter

> **This is a simulated Geiger counter for fun/educational purposes only.**

>  **It does not detect radiation.**

A touch-sensitive Geiger counter simulator on the ESP32-S3.
Touch the cable on Pin 4 to increase the click rate.

## Hardware
- ESP32-S3
- Buzzer/Speaker on Pin 5 (via 100–330Ω resistor recommended)
- Touch cable on Pin 4

## Usage
Flash with Arduino IDE or PlatformIO.
